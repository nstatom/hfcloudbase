Cloud Height Plot Information

Currently the plots can contain up to 168 hours (1 week) of METAR data from standard ASOS/AWOS stations at a given ICAO airport code. Plotted cloud heights are in reference to the airport ground elevation and rounded to the nearest 100 ft. Add the specific airport elevation listed above to get cloud heights in reference to MSL. Clear sky conditions (yellow circles) are plotted at 0 ft elevation and the flight rules category is plotted at a negative elevation arbitrarily for quick referencing (color coded squares). Note that lower level cloudy conditions (especially overcast) can obscure higher elevation clouds from being detected by the airport's ceilometer.

Hourly cloud data ('STD', larger filled icons) are based on standard airport METAR readings that are fetched using the Synoptic Data API. These observations average the past 30 minutes of airport ceilometer data with double weighting for the last 10 minutes of passing sky condition observations. High frequency cloud data ('HF', smaller unfilled icons) are generated from the same METAR stations and fetched using the same API. The resulting HF ceilometer observations may be more prone to overall cloud status mischaracterization in patchy cloud conditions than the hourly METAR data since each HF observation is the result of only a 5 minute collection of vertical point measurements with standard averaging.





Meteorological Plot Information

Wind direction is plotted as 0 by default if the accompanying wind speed is zero.
